IsenFlixe Application Android de gestion de filmographie de films.
L'utilisateur peux noter les films qu'il a vu, ceux qu'il veux voir, gérer sa collection de DVD/Blu-Ray et même voir les films à vendre des autres utilisateurs.

Fonctionnalités :
- Recherche de films par titre, univers, genre et année.
- Film du moment aléatoire à chaque visite de l'accueil.  a
- Catalogue organisé par univers (Disney, Marvel, Pixar, Star Wars, Avatar..)
- Films vus = suivi personnel des films regardés.
- Liste à voir = films à regarder plus tard.
- Collection personnelle = films possédés en DVD/Blu-Ray.
- Marché entre utilisateurs = voir et proposer des films à vendre.
- Profil utilisateur avec gestion du compte, inscription/identification.
- Affiches des films via l'API OMDB.
- Base de donnée Firebase.

Technologies utilisées : 
- Kotlin + Jetpack Compose
- Firebase (Auth + Realtime Database)
- OMDB API pour les affiches
- Retrofit + Coil
